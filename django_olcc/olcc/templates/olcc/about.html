About

Welcome to Oregon Liquor Prices, the unofficial OLCC price list!

You can find the official OLCC product search at oregonliquorsearch.com.

Unfortunately the OLCC doesn't offer an API, but they do publish the current month's price list on their website.

A few years ago I hacked together a quick and dirty script to import that list and display a crude product search on the web. I always meant to get back to the project and add some awesome new features, but hadn't been able until now.

This site is the result of that experimentation. It's a true example of what can be done with a little bit of time and an open data-set. The project is open source and you can view the source code on GitHub.

Thanks for stopping by and stay tuned for more updates.
